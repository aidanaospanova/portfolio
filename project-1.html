Project 1

MaplePath

Immigration Support App

MaplePath is a digital support app designed to help immigrants navigate the Canadian PR process with clarity and confidence. The platform centralizes essential info, personalized checklists, document tools, and consultant booking in one guided experience.

  • RoleUI/UX Designer
  • PlatformMobile App (iOS / Android)
  • FocusClarity, onboarding UX, content structure

Project goals

The Canadian immigration process is complex. MaplePath simplifies this journey by transforming dense information into a guided, supportive experience designed for newcomers.

  • Clarify eligibility criteria and PR pathways.
  • Create personalized checklists.
  • Centralize document uploads, deadlines, updates, and progress.
  • Enable booking with certified partner consultants.
  • Reduce stress using clean UI and supportive microcopy.

Process highlights

1. User research

Interviewed newcomers to identify key barriers such as overwhelming info, confusion about eligibility, and difficulty organizing documents.

2. Information Architecture

Designed the core journey: eligibility → updates → documents → timeline → consultant booking. Created content systems for clarity.

3. UI & Prototyping

Built a calm, supportive interface using soft colours and friendly UX writing. Created high-fidelity prototypes for onboarding, dashboard, and consultant booking.
